Github Pages¶

Open GitHub Pages¶

GitHub provides GitHub Pages services for all repositories. You can open it in the following way:

  1. open a repository

  2. go to repository’s settings

  3. scroll down to find ‘GitHub Pages’

  4. check whether source gh-pages branch
